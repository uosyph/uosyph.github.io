Brain Interpreter

"Brainfk" Interpreter written in Java.   The user is given an array of 30,000 1byte memory blocks, and there are only 8 operators available to use,   ">"   " << /span>"   to move the pointer inside the array,   "+"   "-"   to increase or decrease the value at the pointer's location,   "["   starts a loop,   "]"   ends loop if the block at the pointer's location is zero,   ","   input 1 character,   "."   print 1 character.   Any character besides these operators is ignored by the interpreter and considered a comment.
